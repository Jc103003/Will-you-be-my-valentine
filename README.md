# Will-you-be-my-valentine

A small interactive Valentine’s Day website built as a personal gift.  
The site guides the viewer through a sequence of messages, images, and animations that reveal content over time through user interaction.

---
## Features

- Interactive text progression triggered by user clicks
- Responsive horizontal image carousel with captions
- Animated falling heart background
- Final heart explosion animation after the last message
- Mobile-friendly layout using responsive CSS units
